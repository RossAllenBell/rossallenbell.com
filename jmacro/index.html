JMacro is a java-based application that allows users to write and run macros written in the JMacro scripting language. Using macros written in JMacro, you can simulate input events like pressing keys on the keyboard, moving and clicking the mouse, and even control script flow based on pixel color testing. JMacro is completely free to use.
